# Skybox 변경

Assets/material/에서 Skybox0~5 누르고 인스펙터에서 이미지 변경하면 각 스테이지에 스카이박스 배경 변경됩니다.

스카이박스가 아니라 스카이돔형태의 배경을 사용하려면 돔 형태의 게임 오브젝트를 게임 씬에 따로 배치해야 하는 것 같습니다.
